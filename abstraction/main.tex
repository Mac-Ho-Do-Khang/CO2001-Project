\section*{\Huge Abstract}

Compact interactive devices that blend embedded electronics with 3D-printed enclosures are an increasingly accessible product category, yet commercially available examples remain expensive and offer limited user interaction. This project addresses that gap by designing and developing an integrated handheld device based on the TTGO T-Display ESP32 board housed in a custom 3D-printed enclosure, produced at less than one-quarter of the cost of comparable commercial products.

The development followed a structured four-phase pipeline covering system analysis, mechanical design, electrical integration, and firmware programming. Design for Manufacture and Assembly (DFMA) principles were applied throughout, resulting in a three-component snap-fit enclosure fabricated via Fused Deposition Modelling (FDM) using PLA filament on a Creality Ender 3 V2 printer. The firmware, developed using the Arduino framework, delivers an interactive user interface supporting multiple applications, including the classic games Snake, Pong, and Tetris, a real-time clock display, and additional utilities.

The outcome demonstrates that a functional, interactive, and aesthetically refined handheld device can be realised by combining off-the-shelf embedded hardware with purpose-built additive manufacturing and principled software design, offering a cost-effective, open alternative to proprietary products in the same space.
