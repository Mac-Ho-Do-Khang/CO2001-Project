\subsection{Arduino IDE}

The Arduino Integrated Development Environment (IDE) is a software application that provides a user-friendly interface for programming and uploading code to different microcontroller boards. It is designed to simplify the process of writing, compiling, and uploading code to Arduino devices, making it accessible to both beginners and experienced developers, hence the choice for this project.

\begin{figure}[H]
    \centering
    \includegraphics[width=0.16\textwidth]{components/image/arduinoide_product.png}
    \caption{Arduino IDE logo \cite{arduino_official}}
    \label{fig:arduinoide_product}
\end{figure}

The Arduino IDE supports a wide range of programming languages, including C and C++, and offers a variety of libraries and tools to facilitate the development of projects using Arduino hardware. It is available for multiple operating systems, including Windows, macOS, and Linux, and can be downloaded from the official Arduino website~\cite{arduino_official}.
