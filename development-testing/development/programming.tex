\subsection{Programming}

The final phase of the pipeline covers the software layer that drives the device. Firmware is developed using the Arduino framework together with ESP32-specific libraries, which provide hardware abstraction for the display driver, GPIO management, and power control. The firmware implements the device's core operational logic and manages the content rendered on the 1.14-inch TFT display. The programming phase is carried out iteratively: functionality is built incrementally and validated against the hardware at each step, with issues identified during testing fed back into the firmware until stable behaviour is confirmed.
